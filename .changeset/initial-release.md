Initial release of React Native Image to WebP converter library

- TurboModule implementation with Codegen for iOS and Android
- TypeScript API with validation and 5 presets (balanced, small, fast, lossless, document)
- iOS native module using ImageIO and CoreGraphics
- Android native module using ImageDecoder/BitmapFactory with JNI
- C++ shared code structure for libwebp integration
- Comprehensive documentation and example app
